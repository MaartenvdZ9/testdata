README

This GitHub page contains the datasets used for testing the blending model developed for the Master's thesis: "Improving a Blending Model in the Malt Industry" by Maarten van der Zwaan. This is a thesis submitted in partial fulfillment of the requirements for the degree of Master of Science in Business Analytics and Operations Research

Each dataset is in stored in the ".npz" file format which can be converted to NumPy arrays, a datatype of the popular NumPy library for Python, using the code provided in the file "Load_dataset.py". The code assumes that both "Load_dataset.py" and the datafiles are present in the same folder.

Built on NumPy version: 1.26.4
